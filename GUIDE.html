BEGINNER
GUIDE

Complete guide to launching and monetizing your tools site.

1. Deploy to GitHub Pages (Free Hosting)

  1. Create a GitHub account at github.com
  2. Create a new repository named username.github.io
  3. Upload all your project files to the repository
  4. Go to Settings > Pages and select main branch
  5. Your site is now live at https://username.github.io

2. Google AdSense Setup

  1. Go to adsense.google.com
  2. Sign up with your Google account
  3. Provide your website URL and payment details
  4. Google will review (takes 24-48 hours)
  5. Once approved, add the AdSense code to your site head
  6. Replace placeholder ads with your AdSense ad units
  7. Earnings appear within 48 hours of first clicks

3. Google Search Console & SEO

  1. Visit search.google.com/search-console
  2. Add your website property
  3. Verify ownership using DNS or HTML file
  4. Submit your sitemap.xml file
  5. Request indexing for all pages
  6. Monitor performance in the dashboard

4. Growth Strategy

Week 1: Foundation

  • Deploy site to GitHub Pages
  • Set up Google Search Console
  • Submit sitemap to Google
  • Request indexing for main pages

Week 2-3: Traffic

  • Share on Reddit, ProductHunt, Hacker News
  • Post on social media (TikTok, Instagram, Twitter)
  • Optimize for long-tail keywords
  • Monitor organic traffic in GSC

Month 2-3: Scale

  • Add more tools based on demand
  • Improve site speed and UX
  • Build backlinks through outreach
  • Track AdSense earnings
